## ANSIBLE TOWER
- поддерживается RHL
- используется для управления ANSIBLE через WEB.
- предлагает различные фичи, которые не доступны в командном интерфейсе
    - разграничение по ролям (пользователи, команды)
    - кэширование паролей
    - дизайнер workflow (для создания алгоритма исполнения, связывающего несколько taskk)

## AWX
- open-source решение, на основе которого отрабатываются фичи для TOWER

## Ansible automation Platform
 Концепция объясняющая какие составляющие создают нам среду CI/CD, и внушает мысль, что не очень-то это и правильно использовать свободное ПО, а лучше использовать ПО от REDHAT: console.redhat.com.  Тем не менее большинство компонент, коллекций доступно и на galaxy.ansible.com
